DarkTheme is a simple tool to instantly apply a black and grey based theme to any website. Intended to be used as a bookmarklet, use the examples below to give it a try.

The Basic version of the bookmarklet will change only set colours on the page. Not any background images which may exist. However, the more comprehensive Hardcore version will strip out any background images too. Feel free to copy the encoded Javascript for either version and use it to create your own. However the easiest way to use this is to drag the buttons below each version straight into your browsers favorites bar.
